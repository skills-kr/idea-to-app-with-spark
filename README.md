# GitHub Spark로 아이디어를 앱으로

_생각을 설명하기만 하면 실제 웹 앱으로 만들어 드립니다. 코딩이 필요 없습니다!_

<img width="200" src="https://octodex.github.com/images/sponsortocat.png" align="right">

이 튜토리얼의 예제 Spark 스타터 애플리케이션은 실제 업무를 하는 실제 사람들과의 협업으로 개발되었습니다.

감사의 말씀:

- 종이 접시 설계 엔지니어 - Central Indiana
- 카페 매니저 - Central Oklahoma
- 라크로스 코치 - Northern Massachusetts

## 환영합니다

- **대상**: 문제를 해결하거나 취미를 더 쉽게 만들 아이디어가 있지만, 어디서부터 시작해야 할지 모르는 분
- **배울 내용**: 친구나 동료에게 설명하듯 웹 애플리케이션을 만드는 방법
- **만들 것**: 종이 접시 엔지니어, 바리스타, 라크로스 코치의 맞춤 문제를 해결하는 3가지 웹 애플리케이션 중 1개를 만듭니다. 인터랙티브 기능, 데이터 저장, AI까지 포함됩니다.
- **사전 요구사항**:

  - GitHub Copilot 유료 요금제 (Copilot Pro+ 또는 Copilot Enterprise)
  - 웹 브라우징 및 GitHub 탐색에 대한 기본적인 이해

- **소요 시간**: 이 실습은 1시간 이내에 완료할 수 있습니다.

이 실습에서 다음을 수행합니다:

1. GitHub Spark가 프로그래머뿐만 아니라 모든 사람에게 앱 개발을 가능하게 하는 방법 알아보기
1. 예제 시나리오를 선택하고 첫 번째 웹 애플리케이션 만들기
1. 대화를 통해 앱을 개선하고 커스터마이징하기
1. 직장이나 커뮤니티에 애플리케이션 공유하기
1. 고급 기능과 기반 코드 탐색하기

> [!IMPORTANT]
> GitHub Spark는 현재 공개 프리뷰 상태입니다. 시작하기 전에 접근 권한이 있는지 확인하거나 [요청](https://github.com/features/spark)해 주세요.
